# Turbo Spin Arena

Браузерный прототип аркадного боя волчков на Vite, TypeScript и Three.js.

Игрок управляет волчком на процедурной арене-чаше, разгоняется движением, сталкивается с ботами и старается наносить выгодные удары. RPM работает одновременно как ресурс движения, визуальная скорость вращения и боевое состояние.

## Что уже есть

- Стартовое меню с выбором режима, настройками ботов и выбором элемента.
- Локализация меню на 12 языков: `ru`, `en`, `tr`, `es`, `pt`, `ar`, `id`, `hi`, `vi`, `zh`, `de`, `fr`.
- Четыре элемента: огонь, лед, земля, молния. Пока они влияют на визуальную ауру ультимейта.
- Procedural bowl arena: неглубокая арена-чаша с текстурами, профильным бортиком, плиточной внешней зоной и простым backdrop-окружением.
- Псевдо-гравитация по склону: вниз двигаться легче, вверх сложнее.
- Мягкое сопротивление у белого бортика без жесткой невидимой стены.
- Точки интереса на арене со случайными бонусами.
- FPS counter в углу экрана и первый проход оптимизации рендера.

## Управление

- Мышь: ведите курсор, чтобы задать цель движения игрока.
- Тач: нажмите или ведите палец по экрану.
- ЛКМ: рывок к направлению движения. Кулдаун - 8 секунд.
- ПКМ: активировать ультимейт, если шкала готова.
- Камера зафиксирована на игровом ракурсе.

Волчок не телепортируется к указателю, а следует за ним с ограниченной скоростью, инерцией, зависимостью от RPM и влиянием рельефа арены.

## Текущие механики

- RPM-бой: `currentRPM` и `maxRPM` влияют на движение, восстановление и живучесть волчка.
- Столкновения: урон зависит от скорости удара и зон контакта.
- Зоны контакта: передняя, боковая и задняя зоны дают разные множители атаки и защиты.
- Критические удары: сильный удар передней зоной по уязвимой зоне цели усиливает урон и отбрасывание.
- Dash: рывок на ЛКМ с HUD-кулдауном.
- Ultimate: заряд от урона, полученного урона и пассивного прироста; активация на ПКМ дает 5 секунд ауры и бонус скорости `+10%`.
- Arena bonuses: в точках интереса появляются случайные бонусы скорости, облегчения крита, критического урона, обычного урона и лечения.
- Timed bonuses: временные бонусы показывают aura VFX на владельце и квадратные иконки-таймеры над RPM HUD игрока.
- Heal bonus: при подборе создает зеленую зону лечения с задержкой; зона восстанавливает `maxRPM` всем волчкам внутри, не поднимая `currentRPM` мгновенно.
- Боты: AI выбирает цели, держит дистанцию, атакует и избегает скучивания.
- VFX: ауры, следы движения, искры, вспышки, числа урона, эффекты столкновений, collect-анимации бонусов и зона лечения.
- Vertical layer: есть поля `grounded`, `verticalVelocity`, `heightOffset` для будущих подбрасываний, но полноценные airborne-атаки еще не реализованы.

## Ассеты

- Модель волчка: `public/assets/Spiner.obj`.
- Шрифт проекта: `public/assets/fonts/ZeroCool.woff2`.
- Изображения стартового меню и элементов: `src/img/`.
- JSON-пресеты аур: `src/VFX/`.

## Запуск локально

```bash
npm install
npm run dev
```

Dev-сервер запускается через Vite на `127.0.0.1`.

## Production

Собрать production-версию:

```bash
npm run build
```

Посмотреть production build локально:

```bash
npm run preview
```

## Структура проекта

- `src/main.ts` - сцена Three.js, камера, ввод, HUD, стартовое меню и игровой цикл.
- `src/arenas/` - активная арена, высота поверхности, projection helpers и геометрия арены.
- `src/simulation/` - движение, бой, dash, ultimate, arena bonuses, slope gravity и AI ботов.
- `src/VFX/` - JSON-пресеты визуальных эффектов и runtime VFX для бонусов.
- `src/tornadoVfx.ts` - runtime для аур и вихревых VFX.
- `src/i18n.ts` - словарь меню и определение языка.
- `src/img/` - изображения стартового меню и иконки элементов.
- `public/assets/` - модель волчка и шрифты.
- `index.html` - HTML/CSS оболочка игры.
