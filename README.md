# Turbo Spin Arena

Браузерный прототип аркадного боя волчков на Vite, TypeScript и Three.js.

Игрок управляет волчком на процедурной арене-чаше, разгоняется движением, сталкивается с ботами, наносит урон и использует элементальную ульту.

## Что уже есть

- Стартовое меню с выбором режима, количества ботов, уровня ботов, размера волчка, света и элемента.
- Локализация меню на 12 языков: `ru`, `en`, `tr`, `es`, `pt`, `ar`, `id`, `hi`, `vi`, `zh`, `de`, `fr`.
- Горизонтальная ориентация игры на мобильных устройствах; в портретной ориентации показывается иконка поворота телефона.
- Процедурная арена-чаша с текстурами, бортиком, внешней зоной и backdrop-окружением.
- Движение с инерцией, RPM, dash и влиянием склона арены.
- RPM-бой: `currentRPM` и `maxRPM` влияют на движение, урон, восстановление и живучесть.
- Столкновения с уроном от скорости, зон контакта и критических ударов.
- Deathmatch-состояния: смерть, респавн, временная неуязвимость и таблица игроков.
- Боты с AI: выбор цели, дистанция, атака и антискучивание.
- Ульта на 5 секунд с бонусом скорости `+10%` и элементальной аурой.
- Четыре элемента ульты:
  - огонь усиливает урон и отбрасывание при столкновении;
  - лёд замедляет, замораживает и делает цель хрупкой;
  - молния запускает цепную молнию по ближайшим целям;
  - земля наносит урон по радиусу и подбрасывает цели вверх.
- Бонусы на арене: скорость, облегчение крита, критический урон, обычный урон и лечение.
- Визуальные pickup-шары бонусов через общий пул `StylizedOrbVfx`.
- Временные бонусы с aura VFX и иконками-таймерами над RPM HUD.
- Heal bonus с зоной лечения после подбора.
- VFX: ауры, следы движения, искры, числа урона, hit/crit-hit эффекты, explosion, freeze, beam, rock ripple, collect-анимации и зона лечения.
- Пулы runtime VFX без создания Three.js-ресурсов во время боя.
- FPS counter в углу экрана.

## Управление

- Мышь: вести курсор, чтобы задать цель движения.
- Тач: нажать или вести палец по экрану.
- ЛКМ: dash в направлении движения. Кулдаун — 8 секунд.
- ПКМ: активировать ульту, если шкала готова.
- Мобильные кнопки: dash и ultimate.

Волчок не телепортируется к указателю, а следует за ним с ограниченной скоростью, инерцией, зависимостью от RPM и влиянием рельефа.

- Текстуры VFX: `public/assets/vfx/`.

## Запуск локально

```bash
npm install
npm run dev
```

Dev-сервер запускается через Vite на `127.0.0.1`.

## Production

Собрать production-версию:

```bash
npm run build
```

Посмотреть production build локально:

```bash
npm run preview
```

## Структура проекта

- `src/main.ts` — сцена Three.js, камера, ввод, HUD, стартовое меню и игровой цикл.
- `src/arenas/` — арена, высота поверхности, projection helpers и геометрия.
- `src/simulation/` — движение, бой, dash, ultimate, элементальные скиллы, бонусы, slope gravity, deathmatch и AI.
- `src/VFX/` — runtime VFX, пулы эффектов и JSON-пресеты.
- `src/input/` — мобильное управление.
- `src/rendering/` — instanced-рендер моделей врагов.
- `src/i18n.ts` — словарь меню и определение языка.
- `src/img/` — изображения меню, иконки и текстуры.
- `public/assets/` — модель, шрифт и VFX-ассеты.
- `index.html` — HTML/CSS оболочка игры.
